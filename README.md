# Netflix-Clone

As part of my web development learning journey, I created a Netflix Clone using HTML, CSS, and JavaScript. This project replicates the basic user interface and functionality of Netflix by fetching movie and TV show data from external APIs.

The application uses the TMDB API to display trending movies and TV shows, while the YouTube Search API is used to fetch trailers. The project focuses on building a responsive and user-friendly interface while improving frontend development and API integration skills.

## Features-

- Movie and TV show browsing:  
  Users can explore trending and popular movies and TV shows fetched dynamically using the TMDB API.

- Trailer integration:  
  The project integrates the YouTube Search API to allow users to watch trailers directly from the application.

- Search functionality:  
  Users can search for their favorite movies and TV shows easily.

- Responsive user interface:  
  The design is fully responsive and optimized for desktops, tablets, and mobile devices.

- Dynamic content rendering:  
  Movie data is dynamically loaded and displayed using JavaScript and API calls.

- Netflix-inspired design:  
  The project recreates the visual feel of Netflix with banners, movie rows, thumbnails, and hover effects.

- Frontend development practice:  
  This project demonstrates practical implementation of HTML, CSS, JavaScript, DOM manipulation, and asynchronous API handling.

- Version control and project organization:  
  The project follows proper file structure and Git-based version control practices.
